THE USER IS NOT ALLOWED TO COPY AND PASTE INTO THE ANSWER BOXES.

APP ALSO HAS "DIACHRONIC AI DETECTION." THIS DETERMINES IF THE USER IS COPYING FROM AI BUT CHANGING THE WORDS TO SOUND HUMAN. IN OTHER WORDS, THE APP FLAGS IT IF THE PATTERN OF ERASURES AND REWRITES IS CONSISTENT WITH THE USER'S COPYING (VIA CHANGING THE WORDING OF) AN LLM-GENERATED RESPONSE. 
THE APP LETS THE USER SUBMIT MATERIAL THAT IT FLAGS AS AI-GENERATED. RIGHT NOW, BECAUSE WE ARE AT INITIAL PHASE OF CONSTRUCTION, NO POINTS OFF FOR SUCH MATERIAL.
THE LETS THE USER DO AS MANY PRACTICE HOMEWORKS AND PRACTICE TESTS AS HE WANTS BEFORE ACTUALLY DOING THE REAL HOMEWORKS AND TESTS. 
THE APP GENERATES NEW PRACTICE PROBLEM SETS DE NOVO AD INFINITUM. IT DOES NOT RECYCLE THE SAME OLD PRACTICE PROBLEMS OVER AND OVER. 
IN THE PRACTICE PROBLEM BOX, THERE IS A TOGGLE. WHEN THE TOGGLE IS ON, THE AI-TUTOR IS ON. IT MONITORS THE STUDENT'S WORK; IF THE STUDENT IS RUNNING INTO PROBLEMS, IT GIVES HIM TIPS. THE USER CAN ASK IT QUESTIONS, AND IT WILL RESPONDS VERBALLY. THE USER CAN ASK FOR CERTAIN KINDS OF PROBLEMS, AND THE AI-TUTOR WILL OBLIGE. 
UNLESS DIRECTED OTHERWISE, THE AI-TUTOR GRADUALLY GIVES THE USER HARDER PROBLEMS IF HE IS GETTING ALL THE PRACTICE PROBLEMS RIGHT; AND IF HE IS GETTING THEM WRONG, IT GRADUALLY GIVES HIM EASIER PROBLEMS.
THE AI-TUTOR ALSO TAKES NOTE OF WHAT THE USER IS BEST AT AND WEAKEST AT, AND (UNLESS EXPLICITLY DIRECTED OTHERWISE BY THE USER) THE APP TRIES TO WORK ON THE 
USER'S WEAKNESSES.
IN THE PRACTICE BOX, WHEN THE AI-TUTOR IS TOGGLED OFF, NO SUGGESTIONS ARE GIVEN. THE APP JUST MARKS ANSWERS CORRECT/INCORRECT; IT PROVIDES EXPLANATIONS AS TO WHY THEY INCORRECT WHEN INCORRECT. BUT EVEN WHEN AI-TUTOR IS TOGGLED OFF, THE APP STILL INCREASES DIFFICULTY WHEN USER IS GETTING EVERYTHING RIGHT AND DECREASES IT WHEN USER IS GETTING STUFF WRONG. 
THE APP HAS COMPREHENSIVE ANALYTICS ON THE USER--WHAT HIS PERFORMANCE IS, ON BOTH OFFICIAL AND PRACTICE ASSIGNMENTS. THE APP WILL ALSO (AT THE USER'S REQUEST) GENERATE A REPORT OF WHAT HE IS GOOD (AND HOW GOOD) AND WHAT HE IS A BAD (AND HOW BAD). THE USER CAN SEE THESE ANALYTICS/THESE REPORTS. 
HOMEWORKS ARE NOT TIMED. 
TESTS/EXAMS ARE TIMED.
