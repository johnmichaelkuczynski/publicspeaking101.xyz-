# 🎙️ Podium — Public Speaking Studio (Monorepo)

A pnpm workspace housing **Podium**, a four-week, spoken-first college public speaking course taught, coached, and graded by AI.

- **App:** `artifacts/speak-course` — the student-facing course (served at `/`).
- **API:** `artifacts/api-server` — Express API at `/api` (DB, object storage, AssemblyAI transcription, Anthropic grading, diagnostics).
- **Videos:** `artifacts/speak-course-demo` (product demo) and `artifacts/speak-course-marketing` (promo).

See `replit.md` for the project overview and `artifacts/speak-course/README.md` + `artifacts/speak-course/BLUEPRINT.md` for product and architecture detail.
